A Soma ID is a single record that follows a learner from Grade 1 to Form 4, across any school that runs Soma. If a student transfers, the new school only needs the Soma ID and approval from the previous school's admin — no re-entering years of academic history.
Every Soma product — Portal, Connect, Finance, Browser and Creative Lab — is included from day one for every active student. There are no premium tiers or feature paywalls. See the Pricing page for the full breakdown.
Yes. Results entry comes with grading presets for both ZIMSEC and Cambridge, so admins can switch between them per class or per subject without building a custom grading sheet.
Student and parent data is access-controlled by role, so a teacher only sees their own classes and a parent only sees their own children. The full detail on what's collected and how it's stored is in the Privacy Policy above.
Soma Portal works on the school's local network for day-to-day admin tasks like results entry. Syncing data between schools, and using Soma Connect or Soma Finance, requires an internet connection.
The Parent and Admin guides above cover the standard Soma setup used by every partner school. If your school has custom settings, your admin or our onboarding team can send a school-specific addendum.
